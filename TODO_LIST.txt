GAME CLASS REFACTORING - EX3 FOUNDATION
======================================

Goal: Transform Game class into an abstract base class with NormalGame and LoadedGame as concrete implementations.
This creates the foundation for the recording/replay system (EX3).

PHASE 1: Prepare Base Class
----------------------------
[ ] Make Game destructor virtual (~Game() → virtual ~Game())
[ ] Move Game constructor to protected section
[ ] Test compilation - should still work identically

PHASE 2: Create NormalGame Class
---------------------------------
[ ] Create NormalGame.h (class declaration inheriting from Game)
[ ] Create NormalGame.cpp (implement constructor and handleInput())

[ ] Update main.cpp to use NormalGame instead of Game
[ ] Update Makefile to include NormalGame.cpp and NormalGame.h
[ ] Test compilation with NormalGame - game should work identically

PHASE 3: Make Game Abstract
----------------------------
[ ] Mark handleInput() as pure virtual in Game.h: virtual void handleInput() = 0;
[ ] Remove Game::handleInput() implementation from Game.cpp
[ ] Mark these methods as virtual (with default implementation):

    - virtual void update();
    - virtual void showMainMenu();
    - virtual void showInstructions();
    - virtual void showPauseMenu();
[ ] Test - should compile with NormalGame, Game cannot be instantiated directly
